# 🌡️ Lei de Wien — Radiação de Corpo Negro do Exoplaneta WASP-96 b

Visualização da curva de radiação de corpo negro aplicada ao exoplaneta
WASP-96 b, famoso por ter sido um dos primeiros exoplanetas analisados
pelo Telescópio James Webb (JWST).

## 📌 O que este projeto faz

Plota a curva de Planck para a temperatura de equilíbrio do WASP-96 b
(709 K), identifica o pico de emissão pela Lei de Wien e posiciona o
espectro visível como referência — demonstrando que o planeta emite
predominantemente no infravermelho, fora do alcance do olho humano.

## 🔭 Contexto científico

- **WASP-96 b** é um gigante gasoso a ~1.150 anos-luz da Terra
- Temperatura de equilíbrio: ~709 K (~436 °C)
- Pico de emissão calculado: ~4,09 µm (infravermelho médio)
- A Lei de Wien estabelece que λ_max = 2,898×10⁻³ / T

## 🛠️ Tecnologias utilizadas

- Python 3
- NumPy
- Matplotlib

## 📊 O que o gráfico mostra

- Curva de radiação espectral (Lei de Planck)
- Pico de emissão marcado com coordenadas exatas
- Região do espectro visível destacada para comparação
